# Installation

Prerequisites: 
 * Node LTS
 * Npm LTS

Install elm:

```sh
sudo npm install -g elm &&
sudo npm install elm-github-install -g
```

Get code:

```sh
git clone https://github.com/michie1/wrs_elm &&
cd wrs_elm
```

Install dependencies:

```sh
npm install &&
elm-github-install

```

Run dev mode:

```sh
npm run dev
```

Check [http://localhost:8080](http://localhost:8080)
