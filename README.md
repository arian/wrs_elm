# Installation

Prerequisites: 
 * Node 
 * Npm 

Install elm:

```sh
sudo npm install -g elm
```

Get code:

```sh
git clone https://github.com/michie1/wrs_elm
cd wrs_elm
```

Install dependencies:

```sh
sudo chmod +x install.sh && ./install.sh
```

Run dev mode:

```sh
npm run dev
```

Check [http://localhost:8080](http://localhost:8080)
